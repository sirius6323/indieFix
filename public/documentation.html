indieFix

Project Description

indieFix is a movie web application that will provide users with access to information about indie movies, directors, actors, and genres. Users will be able to create an account, update their personal information, and create a list of movies to watch or a list of their favorite movies.

Tech Stack and Dependencies

Stack: Node.js, Express
Dependencies: express, bodyParser, mongoose, and morgan

Use Case
/ GET NONE A welcome message to indieFix Test GET request and ensure app is working correctly
/movies GET NONE JSON object holding data of all movies containing Title, Description, Genre, Director, and ImageURL Returns a list of all the movies
/movies/:Title GET NONE JSON object holding data of a single movie with a specific title. Movie contains Title, Description, Genre, Director, and ImageURL Returns information of specified movie by title
/movies/Genre/:Name GET NONE JSON object holding data of a specific genre containing its Name and Description Returns data about the movie genre (description) by Name
/movies/Director/:Name GET NONE JSON object holding data of a director containing their Name, Bio, and Birth year Returns data about the specified director (Name, Bio, Birth year)
