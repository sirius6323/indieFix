indieFix

Project Description

indieFix is a movie web application that will provide users with access to information about indie movies, directors, actors, and genres. Users will be able to create an account, update their personal information, and create a list of movies to watch or a list of their favorite movies.

Tech Stack and Dependencies

Stack: Node.js, Express
Dependencies: express, bodyParser, mongoose, and morgan

Use Case
/ GET NONE A welcome message to indieFix Test GET request and ensure app is working correctly
/movies GET NONE JSON object holding data of all movies Returns a list of all the movies
/movies/[Title] GET NONE JSON object holding data of a movie with a specific title Returns information of specified movie by title
/movies/Genre/[Name] GET NONE JSON object holding data of a specific genre Returns data about the movie genre (description) by Name
/movies/Director/[Name] GET NONE JSON object holding data of a director Returns data about the specified director (bio, birth year, death year) by Name
/users POST JSON
{ "Username": "grayFox123",
"Password": "cqc123",
"Email": "boss123@gmail.com"
"FirstName": "Frank",
"LastName": "Jaeger",
"movies" : "[]" } 		JSON object holding info of new user Allows users to register to indieFix
/users/[Username]/movies/[FavoriteMovies] POST JSON
{ "FavoriteMovies": "Bottle Rocket"} 		JSON object holding data of favorite movie to add Adds favorite movie to users favorite list. User will receive a message that the movie was added to their favorite list
/users/[Username]/movies/[WatchList] POST JSON
{ "WatchList": "Before Sunrise"} 		JSON object holding data of movie to add to users watch list Adds movie to the users watch list. User will receive a message that the movie was added to their watch list
/users/[Username]/Email PUT NONE JSON object holding data of users email to be updated Allows users to update their email on their account. Other user account information will be able to be updated in the next release
/users/[Username]/movies/[FavoriteMovies] DELETE NONE JSON object holding data of movie to remove from users favorite list Allows users to remove a movie from their favorite list. User will receive a message that the specified movie was removed from their favorite list
/users/[Username]/movies/[WatchList] DELETE NONE JSON object holding data of movie to remove from users watch list Allows users to remove a movie from their watch list. User will receive a message that the specified movie was removed from their watch list
/users/[Username] DELETE NONE Status message Allows existing user to deregister from indieFix. User will receive a message informing them that they had been removed from indieFix
