indieFix

Project Description

indieFix is a movie web application that will provide users with access to information about indie movies, directors, actors, and genres. Users will be able to create an account, update their personal information, and create a list of movies to watch or a list of their favorite movies.

Tech Stack and Dependencies

Stack: Node.js, Express
Dependencies: express, bodyParser, and morgan

/movies/Genre/[Name] GET NONE JSON object holding data of a specific genre Returns data about the movie genre (description) by Name
/movies/Director/[Name] GET NONE JSON object holding data of a director Returns data about the specified director (bio, birth year, death year) by Name
